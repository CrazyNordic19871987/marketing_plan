## ✨ О проекте

Этот репозиторий содержит интерактивный HTML-документ — **полный маркетинговый план для детского лагеря "ИнноСити" на сезон 2026 года**. План объединяет в себе стратегию, тактику, бюджетирование и операционное управление в едином, визуально насыщенном интерфейсе.

Проект создан, чтобы централизовать все аспекты маркетинговой кампании: от контент-плана и настроек таргета до чек-листов и SWOT-анализа.

### 🎯 Ключевые цели кампании
*   **Заявки:** 300+
*   **Заполняемость:** 250 мест
*   **Бюджет:** 390 000 ₽
*   **Конверсия:** 15%
*   **Стоимость лида (CPL):** < 1 300 ₽

---

## 🚀 Функциональность

*   **Интерактивная навигация:** Удобное боковое меню и верхние вкладки для быстрого переключения между 17 разделами плана.
*   **Красивый дизайн:** Темная тема с неоновыми акцентами (`#00C48C`, `#FFB800`, `#A855F7`), градиентами и современными шрифтами.
*   **Детальная структура:** План разбит на логические блоки для удобного планирования и контроля.
*   **Визуализация данных:** Графики, воронки продаж, диаграммы, прогресс-бары и KPI-карточки для наглядного представления информации.
*   **Готовые шаблоны:** Таблицы для учета смен, контент-матрицы, канбан-доски и трекинга результатов.
*   **Режим печати:** Стилизован под печать (PDF), что позволяет сохранить план в офлайн-документ.
*   **Адаптивность:** Интерфейс подстраивается под мобильные устройства.

## 🗂️ Структура плана (Разделы)

Проект включает в себя следующие разделы для полного управления кампанией:

| Раздел | Содержание |
| :--- | :--- |
| **1. 📊 Обзор** | Главный дашборд: KPI, ключевые даты, общая воронка продаж. |
| **2. 📅 Смены** | График 5 смен, количество мест и статус. Реферальная программа. |
| **3. 👥 Аудитории** | Сегментация: родители (текущие/внешние), ученики, IT-семьи. |
| **4. 📱 Каналы** | Матрица каналов: школы, соцсети (VK, TG, YouTube), контекстная реклама. |
| **5. 🗓️ Календарь** | Понедельный план с целями по заявкам и динамика роста. |
| **6. 📝 Контент** | Контент-матрица по дням недели (Экспертный, Вовлекающий, Продающий). |
| **7. 💰 Бюджет** | Распределение бюджета по месяцам и каналам, структура расходов. |
| **8. ✅ Чек-лист** | Интерактивный список задач по подготовке. |
| **9. 🎯 Офферы** | УТП для разных сегментов, список акций. |
| **10. 📋 Канбан** | Доска задач: To Do, In Progress, Done. |
| **11. 📆 Недели** | Детальный план по неделям с целями и бюджетом. |
| **12. 📊 Метрики** | CPL по каналам, ROI, аналитика эффективности. |
| **13. 🎯 Сценарии** | Консервативный, базовый и оптимистичный прогнозы. |
| **14. 📣 Кампании** | Дневной план постов, Email-последовательности, идеи креативов. |
| **15. 🔍 SWOT** | Анализ сильных/слабых сторон, возможностей и угроз, анализ конкурентов. |
| **16. 📈 Трекинг** | Ежедневный и целевой трекинг KPI. |
| **17. 🔄 Ретаргет** | Сегменты и стратегии для повторного взаимодействия с аудиторией. |
